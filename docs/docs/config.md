# Configurazione minima
___

Non è necessario settare ogni singola riga del file di configurazione.
Di seguito le principali proprietà

#### ITT_URL, ITT_APIKEY

Sono rispettivamente l'indirizzo del tracker e l'api_key dell'utente
Entrambi fondamentali per comunicare con il tracker

#### TMDB_APIKEY, TVDB_APIKEY

Api_key per TheMovieDB e TvDb
Il bot interroga questi due servizi per ottenere l'ID del vostro titolo e trasmetterlo al tracker


#### IMGBB_KEY, FREE_IMAGE_KEY, LENSDUMP_KEY, PTSCREENS_KEY, IMGFI_KEY, PASSIMA_KEY, IMARIDE_KEY

Nello scopo sono tutti uguali.
Il bot dopo aver estratto uno o più screenshots li carica su uno questi host. 
In caso di fallimento tenta il prossimo host

***Non è obbligatorio utilizzare ogni host ma è consigliabile almeno utilizzarne due***


#### QBIT_USER, QBIT_PASS, QBIT_HOST, QBIT_PORT

Come credo sia intuibile queste sono le proprietà per configurare Qbittorrent
