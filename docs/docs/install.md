# Installa Unit3dUp
----

## Dipendenze Windows:

- Scarica e decomprimi FFmpeg da: https://www.ffmpeg.org/download.html
- Aggiungi la cartella di FFmpeg alla variabile d’ambiente PATH (utente)

### Se vuoi anche creare torrents per documenti (PDF)

- Scarica e decomprimi Poppler per Windows da: https://github.com/oschwartz10612/poppler-windows/releases
- Aggiungi la cartella bin al PATH di sistema (esempio: C:\poppler-24.08.0\Library\bin)
- Chiudi e riapri il terminale
- Verifica l’installazione eseguendo:
- **pdftocairo**

## Dipendenze Linux...:

- sudo apt install ffmpeg
- sudo apt install poppler-utils

---
## Installazione valida per linux e windows

- pip install unit3dup

## Aggiorna il bot quando esce una nuova versione

- pip install unit3dup --upgrade
